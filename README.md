# SupportMind AI 🤖

A customer support agent that remembers every customer's history and resolves issues faster.

## Live Demo
👉 https://web-production-ad285.up.railway.app

## Features
- 🧠 Persistent memory across sessions (Hindsight)
- ⚡ Smart model routing by complexity (cascadeflow)
- 🎫 Auto ticket creation
- 💬 Real-time chat UI

## Stack
Python, Flask, Groq, Hindsight, cascadeflow

## Run Locally
```bash
git clone https://github.com/bodigetejasree/supportmind-ai
cd supportmind-ai
python3 -m venv venv && source venv/bin/activate
pip install -r requirements.txt
# Add your keys to .env
python3 app.py
```
